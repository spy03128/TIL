### 대부분의 그리디 문제는 정렬과 동반된다.

### 회의실 배정

회의가 끝나는 시간 순으로 정렬

```python
n = int(input())
meeting = []

for i in range(n):
	s,e = map(int,input().split())
	meeting.append((s,e))
meeting.sort(key=lambda x: (x[1],x[0]))
endtime = 0
cnt = 0
for s,e in meeting:
	if s >=endtime:
		cnt+=1
		endtime = e

print(cnt)
```

```python
import heapq as hq

q=[]
n = int(input())
for _ in range(n):
    s,e = map(int, input().split())
    hq.heappush(q,(e,s))

cnt, end = 0,0
while q:
    e,s = hq.heappop(q)
    if s >= end:
        cnt+=1
        end = e

print(cnt)
```

### 씨름 선수

```python
n = int(input())
arr = []
for i in range(n):
    x,y = map(int,input().split())
    arr.append((x,y))

arr.sort(key=lambda x: (x[0],x[1]),reverse=True)

cnt = 0
largest = 0
for i in range(n):
    if arr[i][1] > largest:
        cnt+=1
        largest = arr[i][1]
print(cnt)
```

```python
n = int(input())
arr = []

arr.sort(reverse=True)
```
