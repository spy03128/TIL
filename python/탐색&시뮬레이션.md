### 회문 문자열 검사

```python
n = int(input())
for i in range(n):
    result = "NO"
    st = input().upper()
    temp = st[::-1]
    if st == temp:
        result = "YES"
    print("#"+str(i+1)+" "+result)
```

```python
n = int(input())
for i in range(n):
    st = input().upper()
    if st == st[::-1]:
        print("#%d YES" %(i+1))
    else:
        print("#%d NO" %(i+1))
```
